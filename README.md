# survival_valix

Valix Lighthouse custom map for [Horizon's Zombie Survival mode](https://github.com/Horizon-Private-Server/horizon-deadlocked-patch/tree/develop/survival).

![thumbnail](HUD/thumb.png)

## Dependencies

1. Docker
2. [Horizon Forge](https://github.com/Horizon-Private-Server/horizon-forge/wiki/Setup-Guide)

## Install Guide

Install into your forge `/Assets/Maps/` folder and then open the `survival_valix.unity` scene in forge.


For example:
```sh
cd ~/horizon-forge/Assets/Maps
git clone https://github.com/Dnawrkshp/survival_valix.git
```
